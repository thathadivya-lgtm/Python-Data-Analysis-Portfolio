# Python Fundamentals

## 📌 Overview

This folder contains foundational Python exercises and notebooks covering core programming concepts. These tasks are part of structured learning and coding practice to build a strong base in Python.

## 🧠 Topics Covered

* Data Structures (List, Dictionary, Set)
* Conditional Statements
* Strings and Tuples
* Loops (for, while)
* Functions

## 📂 Files Included

* `data_structures_and_conditionals.ipynb` → Covers lists, dictionaries, sets, and conditional logic
* `strings_and_tuples.ipynb` → Working with string operations and tuple data structures
* `loops_and_functions.ipynb` → Implementation of loops and function-based logic

## 🎯 Objective

To develop a strong understanding of Python fundamentals and problem-solving skills through hands-on practice.

## 🛠 Skills Gained

* Core Python programming
* Logical thinking and control flow
* Working with built-in data structures
* Writing reusable functions
