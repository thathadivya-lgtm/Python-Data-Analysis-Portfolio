# Taxi Trip Data Analysis and Visualization

## 📌 Project Overview
This project focuses on analyzing and visualizing taxi trip data using Python.  
The dataset used in this project is the built-in `taxis` dataset available in the Seaborn library.

The objective of this project is to perform Exploratory Data Analysis (EDA) to understand trip behavior, fare patterns, payment methods, travel trends, and relationships between different variables using statistical analysis and visualizations.

## 🔍 Key Insights
- Most taxi trips were concentrated during evening and late-night hours.
- Credit card payments were used more frequently compared to cash payments.
- Short-distance trips occurred more often than long-distance trips.
- Fare amount increased as travel distance increased.
- Some trips showed unusually high fare values, indicating the presence of outliers.
- Pickup and drop-off locations showed uneven trip distribution across different zones.
- Correlation analysis revealed a positive relationship between trip distance and total fare amount.
- Boxplots and histograms helped identify skewed distributions in fare and distance columns.
- Heatmap visualization highlighted relationships between numerical variables in the dataset.
- Missing values were identified and handled before performing analysis and visualization.
